**论坛系统**

**项目介绍**：该项目是一个基于 Vue3 + Element Plus + SSM 的前后端分离论坛系统，涵盖话题展示、话题详情、用户个人中心及创作中心四大模块。我独立完成了项目实现的全过程。  

**技术栈**
前端：Vue3、Element Plus、Vue Router、Pinia、Axios、ES6+、WangEditor、SCSS、Vite
后端：SSM（Spring + SpringMVC + MyBatis-Plus）
工具：pnpm、Git

**功能实现**
1. **话题展示模块**  
   - 实现话题与文章列表展示  
   - 支持无限滚动加载更多内容  
   - 关键词搜索功能，并按话题/文章分类展示结果  

2. **话题详情模块**  
   - 展示话题完整信息  
   - 实现点赞、评论、关注作者功能  
   - 集成 WangEditor 富文本编辑器，支持评论编辑  
   - 评论区支持分页或滚动加载  

3. **用户个人中心**  
   - 分类展示用户动态、发布的话题与文章  
   - 个人信息概览  

4. **创作中心**  
   - 话题与文章编辑功能  
   - 草稿箱保存与读取  
   - 富文本编辑支持
